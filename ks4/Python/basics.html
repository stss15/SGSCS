Python: The Basics

What You'll Learn

  • How to create and use variables in Python
  • Basic arithmetic operations (+, -, *, /, //)
  • How to display output with print()
  • How to get user input with input()

Important Things to Note When Writing Your Programs

Python is case sensitive

so it is important that you use the correct case, otherwise your code will not work.

ℹ Variables like name and Name are completely different in Python! Always pay attention to capitalization.

Text values need to appear in speech marks (") but numbers do not.

When naming variables (i.e. values that you want to store data in) you cannot use any dedicated words such as print, input, etc. otherwise your code will not work.

When saving your files do not save them with any dedicated words that Python already uses, such as print, input, etc. If you do this it will not run and you will need to rename the file before it works.

To edit a program you have saved and closed, right-click on the file and select Edit with IDLE. If you just double-click on the file it will only try to run it and you will not be able to edit it.

Example Code

num1 = 93

Set the value of a variable. If the variable doesn't exist, it will be created. A variable is a container for a value that can change while the program is running.

answer = num1 + num2

Adds together num1 and num2 and stores the result in answer.

answer = num1 - num2

Subtracts num2 from num1 and stores the result in answer.

answer = num1 * num2

Multiplies num1 by num2 and stores the result in answer.

answer = num1 / num2

Divides num1 by num2 and stores the result in answer.

answer = num1 // num2

Performs a whole number division (i.e. 9//4 = 2) and stores the result in answer.

print("First line\nSecond line")

The \n character creates a line break in the output.

print("The answer is", answer)

Displays the text "The answer is" followed by the value stored in answer.

print("This is a message")

Displays text enclosed in quotes. The quotes themselves aren't displayed in the output.

textValue = input("Enter a text value: ")

Displays the prompt and waits for user input, storing the text they enter in textValue.

numValue = int(input("Enter a number: "))

Gets user input as text, then converts it to an integer using int(). Integers can be used in calculations.

Interactive Demo

# Variables and basic operations
num1 = 10
num2 = 5

# Basic mathematical operations
sum_result = num1 + num2
difference = num1 - num2
product = num1 * num2
quotient = num1 / num2
whole_division = num1 // num2

# Display results
print("First number:", num1)
print("Second number:", num2)
print("Sum:", sum_result)
print("Difference:", difference)
print("Product:", product)
print("Division:", quotient)
print("Whole Division:", whole_division)

# Demonstrating text output with print
print("Python is a powerful programming language!")
print("You can display multiple lines\nusing the newline character.")

# Demonstrating variable updates
count = 5
print("Initial count:", count)
count = count + 1
print("Updated count:", count)
