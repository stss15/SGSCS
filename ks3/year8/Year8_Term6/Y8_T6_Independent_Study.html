Year 7 Term 6: Independent Study Tasks

Independent Task 1

Starter: Create a list of at least 5 different ways you use computers in your daily life.

Main Activities

  1. Draw a diagram showing how binary numbers are used to represent decimal numbers 0-7.
  2. Create a table converting the decimal numbers 0-10 to binary.
  3. Write a short paragraph explaining how computers use binary to process information.

Plenary: Create a quiz question about binary numbers that you could ask another student.

Independent Task 2

Starter: Write down three examples of how data is stored on a computer.

Main Activities

  1. Research how text is represented using ASCII code and write a brief explanation.
  2. Convert your name to binary code using an ASCII table (which you can find online).
  3. Create a poster explaining how computers store text data.

Plenary: Explain in your own words why computers need a standard way to represent text.

Independent Task 3

Starter: List three different types of media that can be stored digitally.

Main Activities

  1. Research how images are stored digitally and explain what pixels are.
  2. Compare the difference between bitmap and vector images.
  3. Create a simple pixel art image on squared paper, using different colors to represent different binary values.

Plenary: Explain how image resolution affects file size and quality.

Independent Task 4

Starter: Write down what you know about how sound is stored digitally.

Main Activities

  1. Research and explain the concept of sound sampling.
  2. Create a diagram showing how analog sound waves are converted to digital information.
  3. Explain the differences between common audio file formats (e.g., MP3, WAV).

Plenary: Describe how changing the sampling rate affects sound quality and file size.

Independent Task 5

Starter: Make a list of 5 different file types and what they're used for.

Main Activities

  1. Research and create a table of common file extensions and their purposes.
  2. Explain the difference between lossy and lossless compression.
  3. Create a flow chart showing how data is saved and retrieved on a computer.

Plenary: Explain why compression is important when storing digital media.

Independent Task 6

Starter: Create a mind map of all the binary concepts you've learned so far.

Main Activities

  1. Create a guide that explains to a Year 6 student how computers represent different types of data.
  2. Design a poster showing the journey of data from the real world into computer storage and back.
  3. Write a creative story about being a piece of data traveling through a computer system.

Plenary: Reflect on what you've learned about data representation and write down three key points you want to remember.
